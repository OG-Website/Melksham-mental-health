## 🎯 Quick Build Instructions

### Step 1: Build Standalone Executable

**Linux/Mac:**
```bash
./build_app.sh
```

**Windows:**
```cmd
python -m PyInstaller --name "Melksham Mental Health" --windowed --onefile --icon=assets/logo.png --add-data="data;data" --add-data="assets;assets" src/desktop_app.py
```

### Step 2: Find Your Executable

After build completes, look in `dist/` folder:
- `Melksham Mental Health.exe` (Windows)
- `Melksham Mental Health.app` (Mac)
- `Melksham Mental Health` (Linux)

### Step 3: Create Windows Installer (Optional)

1. Download NSIS: https://nsis.sourceforge.io/
2. Right-click `installer.nsi`
3. Select "Compile NSIS Script"
4. Creates: `Melksham_Mental_Health_Installer.exe`

### Step 4: Distribute

- **Windows**: Send `Melksham_Mental_Health_Installer.exe`
- **Mac**: Send `Melksham Mental Health.app`
- **Linux**: Send `Melksham Mental Health`

Others install and run - **NO PYTHON NEEDED!**
