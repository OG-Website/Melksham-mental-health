# 🚀 Quick Start Guide

## Installation (3 minutes)

### Windows: 
1. Download this repository
2. Double-click `setup.bat`
3. Wait for setup to complete (downloads 500+ pieces of content)
4. Run:  `python src\main.py`

### Mac/Linux:
1. Download this repository
2. Open terminal in the folder
3. Run: `chmod +x setup.sh && ./setup.sh`
4. Run: `python src/main.py`

## First Time Use

When you first run the app, it will:
- ✅ Load 500+ pieces of curated mental health content
- ✅ Include all Melksham & Wiltshire resources
- ✅ Have crisis support information ready
- ✅ Generate positive, evidence-based posts instantly

## What You Can Do

### 1. Generate Random Post
Press `1` to get instant positive mental health content

### 2. Generate by Type
Press `2` to choose:  Quote, Story, Advice, Affirmation, etc.

### 3. Generate by Topic
Press `3` to filter by:  Depression, Anxiety, Stress, and 15+ more topics

### 4. View Local Resources
Press `6` to see all Melksham & Wiltshire support services

### 5. Crisis Support
Press `7` for immediate crisis helpline information
