## 📋 Executive Summary

This document outlines the complete scope of work for building the **Melksham Mental Health** website. The website will serve as:

1. **Community Hub** - Mental health resources, support, and information
2. **App Companion** - Download and support for the Post Creator desktop app
3. **E-commerce Platform** - License sales for the Post Creator app (£20 one-time)
4. **Social Presence** - Integration with Facebook and other social platforms

## 💳 Payment Integration

### Stripe Setup

1. **Create Stripe Account**
   - Business type: Individual/Sole trader
   - Country: UK
   - Currency: GBP

2. **Create Product**
   - Name: "Post Creator License"
   - Price: £20.00 (one-time)
   - Description: "Remove Melksham Mental Health branding from posts"

3. **Checkout Flow**
   - Stripe Checkout (hosted)
   - Collect email for license delivery
   - Success redirect to /license/success
   - Cancel redirect to /license/purchase

4. **Webhook for License Generation**
   - Listen for `checkout.session.completed`
   - Generate license key
   - Store in database
   - Email to customer

### License Key System

**Key Format:** `MMH-XXXX-XXXX-XXXX`

**Generation:**
```python
import random
import string

def generate_license_key():
    chars = string.ascii_uppercase + string.digits
    parts = ['MMH']
    for _ in range(3):
        parts.append(''.join(random.choices(chars, k=4)))
    return '-'.join(parts)
```

**Storage:**
- Database table: licenses
- Fields: id, key, email, created_at, activated_at, is_active

**Validation API:**
- Endpoint: POST /api/license/validate
- Input: license_key
- Output: { valid: boolean, message: string }
