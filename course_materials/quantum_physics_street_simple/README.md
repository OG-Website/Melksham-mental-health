## Standard module template used
Each module should include:
1. Narration script (15–20 min spoken)
2. Street-level explanation
3. Scientific explanation
4. Real-world examples
5. Visual description section (for diagrams/slides)
6. Discussion questions
7. Quick knowledge test
8. Teaching version notes
9. Youth/Prison simplified version
