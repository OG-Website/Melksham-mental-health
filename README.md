# Melksham Mental Health - Post Creator 💚

A comprehensive, UK-focused mental health content generator designed to create positive, evidence-based posts for raising awareness and supporting mental wellbeing in Melksham, Wiltshire, and across the UK.

## 🌟 Overview

This tool generates diverse mental health content including:
- ✨ Inspirational quotes from UK mental health professionals
- 📖 Uplifting recovery stories
- 💡 Evidence-based practical advice
- 💚 Positive affirmations
- 📚 Educational content about mental health conditions
- 📍 Local Melksham & Wiltshire resources
- 🆘 Crisis support information
- 🌈 Seasonal mental health support

## 🎯 Features

### Core Functionality
- **Random Post Generation** - Get instant positive mental health content
- **Filtered Content** - Generate by type, topic, or audience
- **Batch Generation** - Create multiple posts at once
- **Local Resources** - Access Melksham and Wiltshire specific support services
- **Crisis Information** - Quick access to emergency mental health support
- **Content Search** - Find specific topics or keywords
- **Statistics** - View content library metrics

## 📦 Installation

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

### Setup

1. **Clone or download this repository**
```bash
git clone https://github.com/wobbob89/Melksham-Mental-Health. git
cd Melksham-Mental-Health
```

2. **No additional dependencies required! **
This program uses only Python standard libraries for maximum accessibility.

3. **Verify data files**
Ensure all JSON files are in the `data/` directory:
- quotes.json
- stories.json
- advice.json
- affirmations.json
- educational. json
- resources.json
- crisis.json
- seasonal.json

## 🚀 Usage

### Running the Application

```bash
python src/main.py
```

### Main Menu Options

```
1.  Generate Random Post - Get instant mental health content
2.  Generate by Content Type - Choose stories, quotes, advice, etc.
3.  Generate by Topic - Filter by specific mental health topics
4.  Generate by Audience - Target specific demographics
5.  Batch Generate Posts - Create multiple posts at once
6.  View Local Resources - Melksham & Wiltshire support services
7.  View Crisis Support - Emergency mental health contacts
8.  Search Content - Find specific topics or keywords
9.  Statistics - View content library metrics
10. Settings - Customize output options
11. Exit - Close the application
```

### Example Usage

**Generate a random post:**
```
Select option 1
View the generated content
Choose to save or generate another
```

**Find anxiety support:**
```
Select option 3 (Generate by Topic)
Choose "Anxiety"
View evidence-based anxiety support content
```

**Access crisis support:**
```
Select option 7
View 24/7 crisis hotlines and local services
```
